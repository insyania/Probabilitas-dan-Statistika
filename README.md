## 📖 About This Repository

This repository contains coursework from the **Probability and Statistics** course at UGM. The course bridges mathematical theory with practical data computing using Python — a critical foundation for data analysis, machine learning, and security data modeling.


## 🧠 Topics Covered

**Probability**
- Sample space, events, and probability rules
- Conditional probability and Bayes' theorem
- Probability distributions (Binomial, Poisson, Normal)
- Random variables (discrete & continuous)

**Statistics**
- Descriptive statistics (mean, median, mode, variance, standard deviation)
- Data visualization (histograms, boxplots, scatter plots)
- Inferential statistics & hypothesis testing
- Correlation and regression analysis
